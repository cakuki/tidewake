## 2026-07-01 — Deep-reading #4: music that *teaches* the battle's shape (onboarding via score)

Web research, new + classic. Sources: adaptive-music combat layering (thegameaudioco, Wikipedia, Splice), Dead Space 2's discrete "fear" layering, and — outside games — music-in-the-classroom "behavior cue" pedagogy (Edutopia). Dedup vs DL#1-3 (vertical layering / bar-clock / stingers / crew chorus).

- **Discrete phase-layers beat a smooth intensity dial for *legibility*.** A continuous calm→combat crossfade reads as "mood," but Dead Space 2 keyed *four distinct layers* to four fear states — a newcomer can hear which state they're in. Our battle epic #135 has real phases (approach / broadside / boarding / etc.) but they're only signalled in the HUD. Giving each phase a **distinct, recognisable musical signature** (not just louder) makes the structure audible: the ear learns the sequence before the fingers learn E/SPACE/X/F/1/2.
- **The bar-clock (DL#3) is what makes phase-cuts feel *composed*, not switched** — pair it with per-phase signatures so each transition lands on a downbeat with a tiny bridge gesture.
- **Cross-connection (outside games — classroom pedagogy):** teachers use a song so students "internalise the length and know when to move to the next task." That's onboarding by *rhythm, not text*. A battle newcomer can learn *when* to act from the phase music's arc long before they know *which* key — the score becomes the tutorial timer.
